Nofollow-check

Checks whether all external Hyperlinks on a given page have rel="nofollow" set. Helps with SEO.

Welcome to Nofollow-Check.

Nofollow-check is a very small JavaScript that can be run to detect any external-pointing hyperlinks on a page which are missing the rel="nofollow" attribute.

Setting rel="nofollow" is beneficial for search engine optimisation.

Getting Started

Simply paste the below code into your browser console:

var s = document.createElement('script'); s.src = 'https://raw.githubusercontent.com/samuelmac/nofollow-check/master/nofollow-check.min.js'; document.querySelector('body').appendChild(s);

Any links missing the appropriate rel tag will then log themselves out into the console for you to see.

Permanent reporting

You can of course link directly to the JavaScript file reference above via a script tag on your page, however this is ill-advised for production websites. Every page log will log information to the console.

Benefits

  • No jQuery required.
  • Vanilla JavaScript.
  • Just requires any new browser (Chrome, Firefox, IE9+) with a developer console.
